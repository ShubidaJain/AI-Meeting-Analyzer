# 🧠 AI Meeting Analyzer

## 💡 Problem
Meetings generate a lot of unstructured information. Important decisions and action items are often lost.

## ✅ Solution
An AI-powered system that extracts:
- 📌 Decisions
- ✅ Action Items  
from meeting discussions automatically.

## ⚙️ Tech Stack
- Frontend: React
- Backend: FastAPI
- AI: Groq API
- Deployment: Vercel + Render


## 🛠 Setup Instructions

### Backend 
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm start

## 📸 Features
Login / Signup
AI Analysis Dashboard
History Tracking
Copy Results
